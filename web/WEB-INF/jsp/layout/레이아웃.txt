JSP 페이지 모듈화
일반적으로 웹 페이지는 동일한 상/하단 영역과
메뉴에 따라 달라지는 메인영역으로 구성되어있다.

이런 경우 JSP액션 태그를 이용해서 웹 페이지를 모듈화 시키면
코드가 중복되는 것을 방지할 수 있다.

즉, 동일한 상/하단 영역은 jsp:include액션태그로 구성하고
메뉴에 따라 메인영역에 보여질 부분만 jsp 페이지로 지정하는 방식을 사용한다.
=> 탬플릿 사용

포워드 vs 리다이렉트

리다이렉트 redirect : 클라이언트에게 지정된 url로 재요청하도록 지시
    ex) 고객이 상담원에게 전화 -> 특정번호로 다시 하라고 안내

포워드 forward : 클라이언트의 요청을 받은 jsp가 직접 처리하지 않고 다른 jsp에게 처리하라고 지시
    ex) 고객이 상담원에게 전화 -> 상담원이 문의사항을 몰라 팀장에게 물어보고 응답
